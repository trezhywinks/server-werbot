Welcome to WineWinks



WineWinks:

A chatbot created for you. winewinks downloads reels from Instagram, videos from Youtube, songs, creates stickers answers some messages, views status and messages.


Installation (Kali Linux)

Github link
cd werbot
bash install.sh
werbot -s --server



What is needed?

You need a terminal, you need Kali Linux to run this Chatbot. with the command "bash install.sh" You install all dependencies.




Connect! Scanning qrcode

Use your device to scan the qrcode that appears on the terminal.
